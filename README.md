# react-config

## Install

This is functional programming, hooks only React Boilerplate, which I made because I was bored.

Enjoy... or not, who cares.

> 1 step

```bash
git clone https://github.com/timmilesdw/react-config.git
```

> 2 step 

```bash
cd react-config
```

> 3 step 

```bash
yarn
```

> 4 step

```bash
yarn start
```
